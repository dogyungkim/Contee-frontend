### Contee 서비스 – “찬양 추가” 기능 간단 설계서 (MVP 버전)

**목표**  
사용자가 콘티 편집 화면에서 **새로운 찬양**을 만들거나 **기존 찬양**을 불러와서 바로 추가할 수 있게 함.  
모든 과정이 **한 화면 안에서 자연스럽게** 이루어지며, 리더가 “아, 이 찬양 넣고 싶어” 하는 순간에 **지체 없이** 행동할 수 있도록 설계.

#### 1. 화면 위치 및 진입점
- 현재 화면 유지

#### 2. 버튼 상세 설계

| 버튼 이름             | 크기/색상/아이콘                  | 클릭 시 동작                                                                                   | 언제 누르는가? (사용자 마음)                     |
|-----------------------|-----------------------------------|------------------------------------------------------------------------------------------------|-------------------------------------------------|
| **+ 새 찬양 만들기**  | 중간 크기, 보조 색상 (회색/연보라), ✨ 반짝이 아이콘 | 1. 곡 리스트 맨 아래에 **빈 곡 카드** 즉시 생성<br>2. 바로 입력 모드로 전환 (포커스 자동)     | “이번 주에 처음 부르는 신곡이야, 바로 만들어 넣자!” |
| **+ 팀 찬양 불러오기**| 가장 크게, 주요 색상 (파랑/초록), 🔍 또는 📂 아이콘 | **전용 Dialog(모달)** 열림<br>· 탭 1: 팀에서 자주 쓰는 찬양 (기본)<br>· 탭 2: 전체 찬양 라이브러리 | “지난번에 썼던 그 찬양 다시 넣고 싶어” 또는 “팀에 있는 곡 중에서 골라야지” |

#### 3. “새 찬양 만들기” 흐름 (빈 곡 카드 생성)

1. 버튼 클릭 → 곡 리스트 맨 아래에 **빈 곡 카드** 생성 (회색 톤 placeholder)
2. 카드 내부 입력 필드 (한 번에 다 보이게):
   - **곡 제목** (필수, 큰 입력창, placeholder: “찬양 제목 입력”)
   - **아티스트** (선택)
   - **키** (드롭다운: C, D, E♭ 등 + 커스텀 입력)
   - **BPM** (± 버튼 + 숫자 입력)
   - **Song Form** (간단 파트 추가 UI: + 버튼으로 Verse, Chorus 등 추가 → 드래그로 순서 변경)
   - **악보/음원/유튜브 링크** (URL 입력창 + 미리보기 버튼)
   - **노트** (멀티라인 입력)
3. 입력 중 실시간으로 카드 색상 정상화 (회색 → 흰색)
4. “저장” 버튼 클릭 시:
   - 팀 곡 DB(`team_songs`)에 자동 등록
   - 콘티에 바로 추가 (`conti_songs`)
   - 토스트: “새 찬양이 콘티에 추가되었습니다”

#### 4. “팀 찬양 불러오기” Dialog 설계

- **Dialog 크기**: 화면 70~80% 높이, 모바일에서도 스크롤 편함
- **상단**: 큰 검색바 (곡명 · 아티스트 · 키 · BPM 검색)
- **탭 2개** (기본: 팀에서 자주 쓰는 찬양)

| 탭 이름                  | 내용                                                                 | 정렬 옵션                     |
|--------------------------|----------------------------------------------------------------------|-------------------------------|
| **팀에서 자주 쓰는 찬양** | 팀 내 team_songs 중 실제 콘티에 사용된 적 있는 곡들 (최근/자주 순) | 최근 사용순 / 자주 사용순     |
| **전체 찬양 라이브러리** | 팀 전체 team_songs + 공식 songs DB에서 팀이 아직 안 쓴 곡도 검색 가능 | 제목순 / 아티스트순 / key 순|

- **곡 카드 미리보기** (Dialog 안에서도 한눈에):
  - 제목 (굵게)
  - 키 · BPM · Song Form 요약 (V → C → Br → C×2)
  - 마지막 사용 날짜 (팀 탭에서만)
- **선택 방식**:
  - 한 곡 클릭 → 즉시 콘티에 추가 + Dialog 자동 닫힘
  - 멀티 선택 (체크박스) → “선택한 곡 모두 추가” 버튼
- **팀에 없는 곡 선택 시**:
  - “이 찬양은 팀에 아직 없어요. 팀 곡으로 추가하시겠어요?” 확인
  - “추가하고 불러오기” → 자동으로 team_songs 등록 후 콘티 추가

#### 5. 핵심 UX 원칙
- **한 화면 완성형** → Dialog 닫히면 바로 콘티 전체가 보임
- **인지 부하 최소** → “새 찬양”은 바로 입력 시작, “기존 찬양”은 검색 중심
- **실수 방지** → 빈 카드 입력 중 “취소” 버튼 제공 (카드 삭제)
- **모바일 최적화** → 하단 바 버튼 터치 영역 크게, Dialog 스크롤 부드럽게
